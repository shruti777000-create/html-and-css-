#  Sample College website

This is a sample college website. It showcases the development of a simple multi-page website using HTML5 and CSS3, demonstrating fundamental web development concepts including forms, styling, and responsive design.

## Pages Included

- **index.html**: The main homepage of the website, serving as the entry point.
- **Login.html**: A user login page with form elements for authentication.
- **Applynow.html**: An application submission page with input fields for user applications.

## Technologies Used

- **HTML5**: For structuring the web pages and content.
- **CSS3**: For styling the pages, including layouts, colors, fonts, and responsive elements.

## How to View the Website

1. Clone or download this repository to your local machine.
2. Open any of the HTML files (e.g., `index.html`) in a modern web browser.
3. Navigate between pages using links or by opening the respective HTML files.

No server setup is required as this is a static website.

## Features

- Clean and simple design
- Form validation (basic HTML5 validation)
- Responsive layout elements
- Image integration for visual appeal
